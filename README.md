# Cordel Moderno

Pequena página estática que exibe o cordel "Cordel Moderno" com estilo responsivo.

## Como visualizar
Abra o arquivo principal no navegador:
- [index.html](index.html)

## Arquivos principais
- Página HTML: [index.html](index.html)  
- Estilos: [estilo/style.css](estilo/style.css) — variáveis de fonte: [`--font1`](estilo/style.css), [`--font2`](estilo/style.css), [`--font3`](estilo/style.css)  
- Texto do cordel: [`Cordel Moderno`](cordel-moderno.txt)  
- Imagens de fundo: [imagens/](imagens/) — usadas por [`section#img01`](estilo/style.css) e [`section#img02`](estilo/style.css)  
- Licença: [LICENSE](LICENSE)  
- Configuração Git: [.gitattributes](.gitattributes)

## Personalização rápida
- Altere o poema em [`cordel-moderno.txt`](cordel-moderno.txt) e/ou em [index.html](index.html).  
- Modifique cores/ fontes em [estilo/style.css](estilo/style.css).  
- Substitua as imagens em [imagens/](imagens/) para trocar os fundos (`section#img01`, `section#img02`).
